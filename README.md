# IPL 2025 Venue Data Analysis

## 📌 Project Objective
To analyze IPL 2025 match and venue data to identify performance patterns and stadium trends.

## 🛠 Tools Used
- Microsoft Excel
- SQL
- Data Cleaning Techniques
- Pivot Tables & Charts

## 📊 Analysis Performed
- Total matches played at each venue
- Win percentage by venue
- Toss decision impact
- High scoring vs low scoring stadiums
- Team performance across different grounds

## 📈 Key Insights
- Certain venues favor chasing teams.
- Toss decision impacts match outcome in specific stadiums.
- Some venues consistently produce high scores.

## 🎯 Conclusion
This project demonstrates practical data analysis skills using real-world cricket data.
